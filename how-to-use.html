Date & Time Clocks

How to use the clocks

The clock itself can be found here (or here with white text) and can be added to any site using the iframe code here:

<iframe src="https://lordgnomembe.github.io/today-clock/" class="class-name">
</iframe>

or this code for white text:

<iframe src="https://lordgnomembe.github.io/today-clock/index-white-text.html" class="class-name">
</iframe>

It can then be styled using the following code:

.class-name {
width: 200px;
height: 120px;
border: 3px solid black;
border-radius: 20px;
}

Styling the background of the iframe changes the background of the clock, like this:

.class-name {
width: 200px;
height: 120px;
border: 3px solid black;
border-radius: 20px;
background: red;
}

As such it can be given an image as background and any of the associated stylings:

.class-name {
width: 200px;
height: 120px;
border: 3px solid black;
border-radius: 20px;
background-image: url("https://wosu.org/wp-content/uploads/2015/10/Ball-of-Sun.jpg");
background-size: cover;
}

Likewise it can be given a gif background:

.class-name {
width: 200px;
height: 120px;
border: 3px solid black;
border-radius: 20px;
background-image: url("https://media0.giphy.com/media/fqfPOiwKagbluB8Ors/giphy.gif");
background-size: cover;
}
